## How it works

| 1. Python Basics | Syntax, functions, OOP |
| 2. Flask Web Development | HTTP, templates, forms |
| 3. FastAPI & REST APIs | REST, Pydantic, CRUD |

| 5. Capstone Projects | Final projects |

This matches the [DE Training Program](https://github.com/users/vikas-yadav-neo/projects/1) layout — one view with module sections, not Todo/In Progress/Done.
