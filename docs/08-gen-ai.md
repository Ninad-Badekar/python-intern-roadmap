# Module 9: Gen AI

**Goal:** Work with LLMs practically — prompting, embeddings, and RAG — and know when to use fine-tuning.

**Duration:** 2–3 weeks

**Prerequisites:** [Module 8 — Deep Learning](./07-deep-learning.md) (helpful but not mandatory for prompting/RAG)

---

## Topics & checklist

### LLM basics
- [ ] What an LLM is (next-token prediction, high level)
- [ ] Tokens, context window, temperature, top-p
- [ ] System vs user prompts
- [ ] Hallucinations — causes and mitigations
- [ ] Safety / responsible use basics

### Prompt engineering
- [ ] Clear instructions and output formats
- [ ] Zero-shot and few-shot examples
- [ ] Chain-of-thought prompting (when useful)
- [ ] Structured outputs (JSON)

### RAG & embeddings
- [ ] What embeddings are
- [ ] Similarity search (cosine similarity intuition)
- [ ] Document chunking strategies
- [ ] RAG pipeline: retrieve → augment prompt → generate
- [ ] Vector DB awareness (FAISS / Chroma / Pinecone)
- [ ] Prompting vs RAG vs fine-tuning — tradeoffs

### Hands-on
- [ ] Call an LLM API from Python
- [ ] Build a mini RAG over your own PDF/markdown docs
- [ ] Evaluate answers qualitatively (groundedness)

---

## Mini project

**Doc Q&A bot**
- [ ] Ingest a small document set
- [ ] Retrieve relevant chunks
- [ ] Answer questions with citations
- [ ] README with setup and example queries
