# Module 7: Deep Learning

**Goal:** Understand neural networks and common architectures at an intern level, and train a simple model in PyTorch or Keras.

**Duration:** 2–3 weeks

**Prerequisites:** [Module 6 — Machine Learning Basics](./04-data-science.md)

---

## Topics & checklist

### Neural network fundamentals
- [ ] Neurons, layers, weights, bias (intuition)
- [ ] Activation functions: ReLU, sigmoid, softmax
- [ ] Loss functions (MSE, cross-entropy)
- [ ] Forward pass + backpropagation (high level)
- [ ] Optimizers: SGD, Adam (awareness)
- [ ] Epochs, batches, learning rate
- [ ] Overfitting controls: dropout, early stopping, regularization

### Architectures (intro)
- [ ] CNNs for images — convolution & pooling intuition
- [ ] RNNs / LSTMs for sequences — intuition
- [ ] Attention & Transformers — high-level idea
- [ ] Transfer learning (pretrained models)

### Framework practice (pick one)
- [ ] Install PyTorch **or** TensorFlow/Keras
- [ ] Build a simple classifier (MNIST / Fashion-MNIST)
- [ ] Plot training/validation loss
- [ ] Save and load a model

---

## Mini project

- [ ] Train an image or text classifier
- [ ] Report accuracy + confusion matrix
- [ ] Short write-up: architecture choices and results
