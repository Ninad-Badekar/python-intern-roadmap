# Contributing

Thanks for helping improve the Python intern roadmap!

## Ways to contribute

- Fix typos or unclear explanations
- Add vetted learning resources (official docs preferred)
- Suggest new practice exercises or capstone ideas
- Improve examples for clarity

## Pull request process

1. Fork the repository
2. Create a branch: `git checkout -b fix/typo-in-flask-doc`
3. Make your changes
4. Commit with a clear message
5. Open a pull request describing what you changed and why

## Style guidelines

- Keep language simple — interns may be early in their careers
- Prefer links to official documentation over blog spam
- Include code examples that run on Python 3.11+
- Don't add paid course affiliate links

## Questions

Open an issue with the `question` label if you're unsure whether a change fits.
